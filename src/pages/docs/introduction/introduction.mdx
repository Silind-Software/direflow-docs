## Why Web Components?
Web Components is a browser standard that was added by WC3 (World Wide Web Consortium).  
They give us all the power that a modular, atomic, component-based design does, but in a framework-agnostic way that enables us to run encapsulated components directly in the browser as custom HTML elements.
  
With Web Components you can consume individual components without depending on any frameworks like React or Angular.

## Why React?
Even though Web Components can be build entirely with vanilla JavaScript, it's pretty handy to use a library like React to build the UI.  
React is one of the most popular JavaScript libraries for building UIs and single-page application, and is well known for its fast performance and easy-to-learn nature.
